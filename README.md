# PRODIGY_DS_01 - Data Visualization

## Objective

Create a bar chart and histogram to visualize the distribution of categorical and continuous variables using Python.

## Project Description

This project was completed as part of the **Prodigy InfoTech Data Science Internship (Task 01)**.

The notebook demonstrates how to visualize data distributions by:
- Creating a sample dataset containing **Age** and **Gender**.
- Plotting a **bar chart** to show the distribution of gender.
- Plotting a **histogram** to show the distribution of age.

These visualizations help understand the frequency and distribution of different variables in a dataset.

## Dataset

A sample dataset containing the following attributes:

- Age
- Gender


## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations

### 1. Gender Distribution

- Created using a **Bar Chart**
- Shows the number of males and females in the dataset.

### 2. Age Distribution

- Created using a **Histogram**
- Shows how ages are distributed across the dataset.

## Steps Performed

1. Imported required Python libraries.
2. Created a sample dataset.
3. Loaded the dataset into a Pandas DataFrame.
4. Generated a bar chart for Gender distribution.
5. Generated a histogram for Age distribution.
6. Customized the charts with titles and labels.

## Results

- Successfully visualized the distribution of a categorical variable (Gender).
- Successfully visualized the distribution of a continuous variable (Age).
- Demonstrated basic data visualization techniques using Matplotlib and Seaborn.


## Conclusion

This project demonstrates how graphical visualizations can help understand the distribution of categorical and continuous variables. Bar charts effectively represent category frequencies, while histograms help analyze the spread and pattern of numerical data.
